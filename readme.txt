== Description ==

This is a simple plugin for Wordpress to test if the visitors ISP supports DNSSEC. The result is displayd in a widget. This plugin is totally relying on the code from Loopia AB, www.loopia.se. For more information (in Swedish) see http://blogg.loopia.se/2009/02/23/snalla-stjal-var-dnssec-kod/ .

You are free to do any changes in the code so it fits your theme but please let me, markus(at)liljedahl.me know so I maby can do a global update.

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. Activate the widget and edit the title and texts if you want
